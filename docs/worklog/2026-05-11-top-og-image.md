## 技術的な学び

- Astro のルーティングで、同一ディレクトリ内に静的ルート (`index.png.ts`) と動的ルート (`[id].png.ts`) が共存する場合、静的ルートが優先される。`getStaticPaths` を持つ動的ルートと衝突しないので、トップ用エンドポイントは単純に `src/pages/og/index.png.ts` を置くだけで OK
- satori で OG 画像を生成する場合、`<img>` の `src` には base64 data URL を使うのが確実。ファイルパスを使うと resvg 側で読めないことがある
- X (Twitter) Card のキャッシュは URL 単位。`og:image` の URL にクエリパラメータを追加することで Twitterbot に「別 URL」と認識させて再フェッチさせるのが X 公式ドキュメント記載の workaround
- 公式 Card Validator (`cards-dev.twitter.com/validator`) は 2022〜2023 年に廃止済み。サードパーティの validator (SocialRails, OpenTweet など) が代替
